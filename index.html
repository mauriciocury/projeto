Sobre

Desenvolver um sistema de uma chave de competição voltado para o evento de Sumô de Robôs organizado pela Descomplica Uniamérica.

O sistema deve ser capaz de :

  1. Adicionar e remover participantes. Um participante inscrito deve conter nome, foto (opcional) e integrantes do grupo(opcional).
  2. Gerar aleatoriamente uma chave com os participantes de uma competição.
  3. Avançar a chave de competição ao informar o vencedor de uma partida.
  4. Inserir ou não uma rodada de repescagem conforme solicitação do usuário.
  5. Armazenar histórico de partidas do torneio e permitir que o usuário volte partidas e, também, edite o resultado de uma partida.

Resultados esperados: O sistema completo e operante capaz de rodar dentro de um navegador web.
